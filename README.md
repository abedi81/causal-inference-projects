# Causal Inference Projects Portfolio

This repository contains a collection of projects focused on causal inference, experimental design, and policy evaluation using real-world and simulated data.

The goal is to demonstrate practical and theoretical understanding of how to estimate causal effects using modern data science and econometric methods.

---

## Key Topics Covered

- A/B Testing and Experiment Design  
- Power Analysis and Statistical Inference  
- Difference-in-Differences (DiD)  
- Fixed Effects Models  
- Synthetic Control Methods  
- Regression Discontinuity (RDD)  
- Instrumental Variables (IV)  
- IPW and G-Computation  
- Treatment Effect Heterogeneity
